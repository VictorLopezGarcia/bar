AppBar

Proyecto de gestión de pedidos para un bar/restaurante de alta gama, desarrollado con Flutter.

Descripción

Esta aplicación permite gestionar los pedidos de los clientes de manera eficiente y visualmente atractiva. Diseñada con un enfoque "Premium", ofrece una interfaz oscura y elegante para la creación y seguimiento de comandas.

Tecnologías Usadas

  • Flutter: Framework principal para el desarrollo de la UI.
  • Dart: Lenguaje de programación.
  • Arquitectura MVVM: Separación clara entre la lógica de negocio y la interfaz de usuario.
  • Material Design: Uso de componentes y temas de Material 3 con un esquema de colores personalizado.

Características Principales

  • Gestión de Pedidos: Visualización de pedidos activos en la pantalla principal.
  • Creación de Comandas: Formulario intuitivo para añadir nuevos pedidos.
  • Selección de Productos: Selección múltiple de productos gourmet desde una lista predefinida.
  • Resumen de Pedido: Vista detallada del pedido antes de confirmar.
  • Diseño Premium: Interfaz moderna con tema oscuro (ThemeData.dark) y animaciones fluidas.
